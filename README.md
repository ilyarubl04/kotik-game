# 🐱 Котик: открытый мир

2D-игра, которую папа делает вместе с 7-летним сыном. Один HTML-файл, без серверов и баз данных — открывается прямо в браузере и играется на компьютере, планшете и телефоне.

**▶️ Играть онлайн:** https://kotik-game-rublevsky.netlify.app

## Что внутри
- Открытый мир из 9 биомов: 🌲 Лес → 🏜️ Пустыня → 🏖️ Пляж → 🏙️ Город → ❄️ Снега → 🌋 Вулкан → ☁️ Небеса → 🌌 Космос → 🦇 Пещера → 🏰 Замок
- День и ночь, ночью выходят зомби 🧟
- Сбор ресурсов 🪵🧱 и **строительство своего замка** (на блоки можно запрыгивать!)
- Супер-силы: 🥛 щит, ⭐ супер-режим, 🐟 +жизнь
- Кнопки в небе: 🐉 дракон, 🐲 покататься на драконе, 🙏 Бог, 🏰 подсказки
- Бомбы из домов 💣, воины с катанами 🥷🗡️, финальный босс 👑
- Квест: найди 🌟 9 звёзд (по одной в каждом мире)
- Сохранение прогресса, пауза (Esc), страница прогресса

## Управление
**Компьютер:** ← → ходить · Пробел прыжок (2 раза) · Shift рывок · B строить · M музыка · Esc пауза
**Планшет/телефон:** экранные кнопки появляются автоматически.

## Технологии
Чистый HTML + Canvas + JavaScript. Звук — Web Audio API. Сохранения — localStorage. Базы данных нет — игра полностью статичная, хостится на любом статичном хостинге.
